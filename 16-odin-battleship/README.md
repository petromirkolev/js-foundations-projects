# Battleship (Vanilla JavaScript)

Browser implementation of the classic Battleship game built with plain JavaScript, HTML, and CSS.

The focus is on **game state**, **board logic**, and **DOM rendering** without frameworks.

---

## Features

- 10Ã—10 player and computer boards
- Random fleet generation for both sides
- Turn-based play (player vs. computer)
- Tracks hits, misses, and sunk ships
- Status text + simple action log

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

No build tools or frameworks.
